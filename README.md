# 🏥 Health-Genix

Health-Genix is a web-based healthcare awareness platform designed to promote knowledge about diseases and healthy living.  
It includes interactive features like a symptom checker, quizzes, and health data visualization to engage users and spread awareness.

---

## 🚀 Features

- 🩺 Symptom Checker (basic health guidance)
- 🧠 Health Awareness Quiz
- 📊 Health Data Charts (visual insights)
- 📚 Disease Awareness Information
- 🎨 Attractive and responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

- index.html → Home page  
- quiz.html → Health quiz  
- symptoms.html → Symptom checker  
- charts.html → Data visualization  
- style.css → Styling  
- images/ → Assets (if any)

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open index.html in any browser  

---

## 🎯 Purpose

This project aims to:
- Promote awareness about common diseases  
- Help users understand symptoms  
- Encourage preventive healthcare practices
